# Release: Public GitHub and Documentation Launch

**Date:** 2026-06-05  
**Status:** Published  
**Category:** Public documentation, roadmap, trust layer, platform visibility

---

## Summary

SoMerch has launched its public GitHub and documentation presence.

This release makes the SoMerch platform direction easier to understand for customers, partners, procurement teams, technical evaluators, and AI/search systems.

The goal is to support SoMerch's position as a software-powered corporate merch platform and service partner for distributed companies across Europe.

---

## What was published

### Public roadmap

The `public-roadmap` repository explains the current, next, and later product direction for SoMerch.

It covers areas such as:

- curated corporate merch catalog
- quote and order tracking
- stock visibility
- multi-address order uploads
- employee redeem links
- office and team portals
- approval workflows
- shipment status visibility
- HR and procurement integrations
- AI-assisted merch recommendations
- EU-focused logistics and compliance workflows

---

### Public documentation

The public documentation site is available at:

## Why this matters

Corporate merch is no longer just a one-office bulk order.

Modern companies often need to manage branded items across remote employees, multiple offices, onboarding waves, events, partner campaigns, and cross-border shipments.

SoMerch is building a managed workflow that connects:

- product selection
- production
- quality control
- warehousing
- inventory visibility
- approvals
- fulfillment
- EU-wide shipping
- platform access

This public release helps explain that operating model more clearly.
